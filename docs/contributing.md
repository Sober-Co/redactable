# Contributing

We welcome contributions!

## How to Contribute
- Open an issue for bugs/ideas.
- Submit PRs following the style guide.

## Development Setup
```bash
git clone https://github.com/Sober-Co/redactable.git
cd redactable
pip install -e ".[dev]"
```

## Code Style

- Python 3.13+
- Type hints, docstrings
- `ruff` linting
- `pytest` for tests
